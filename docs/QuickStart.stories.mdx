  </div>

## How to use

### Setup

Before using `@liveperson-design-system/icons` or `@liveperson-design-system/components`,
you have to add the following script inside `head` tag

<TabsState>
  <div id="icons" title="Icons">

```html
<script src="https://unpkg.com/@liveperson-design-system/icons/vendor.js"></script>
```

  </div>
  <div id="components" title="Components">

```html
<script src="https://unpkg.com/@liveperson-design-system/components/vendor.js"></script>
```

  </div>
</TabsState>

### Usage

You can use any LPDS component either in a browser or with a front-end framework

<TabsState>
  <div id="browser" title="Browser">

```html
<html>
  <head>
    <script src="https://unpkg.com/@liveperson-design-system/components/button.js" />
  </head>

  <body>
    <lp-button labal="Button" />
  </body>
</html>
```

  </div>
  <div id="framework" title="Framework (React)">

```jsx
import React from 'react';

import '@liveperson-design-system/components/button';

const App = () => {
  return (
    <lp-button label="Button" />
  );
};

export default App
```

  </div>
</TabsState>

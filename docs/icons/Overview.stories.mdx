# Icons

Each icon is a separate module.
It means that you have to import a specific module for the icon that you are going to use

<TabsState>
  <div id="js" title="JavaScript">

```jsx
import '@liveperson-design-system/icons/arrow-down';

<lp-arrow-down-icon />
```

  </div>
  <div id="browser" title="Browser">

```html
<script src="https://unpkg.com/@liveperson-design-system/icons/arrow-down.js"></script>

<lp-arrow-down-icon />
```

  </div>
</TabsState>

## Default selected

You can easily set the default selected button by adding `selected` class to the button item

<Preview>
  <lp-button-group>
    {Array(5).fill(null).map((item, index) => (
      <button key={index} className={index === 2 ? 'selected' : undefined}>{`Button ${++index}`}</button>
    ))}
  </lp-button-group>
</Preview>

<Preview light>
  <lp-button-group theme="light">
    {Array(5).fill(null).map((item, index) => (
      <button key={index} className={index === 2 ? 'selected' : undefined}>{`Button ${++index}`}</button>
    ))}
  </lp-button-group>
</Preview>

```html
<lp-button-group>
  <button>Button 1</button>
  <button>Button 2</button>
  <button class="selected">Button 3</button>
  <button>Button 4</button>
  <button>Button 5</button>
</lp-button-group>
```

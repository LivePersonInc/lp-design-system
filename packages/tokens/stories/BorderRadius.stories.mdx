# Border radius

Border-radius are used on most components that are used as containers.
Nested components should be smaller than the parent component.

## Mixins

- `border-radius($border-radius)` - returns `border-radius` style
  - `$border-radius` - available size or just a value

### JS

- `borderRadius: (size: BorderRadius | string | number): string`

## CSS helpers classes

### Classes

To apply `border-radius` style just add `lpds-border-radius-{size}` class to the element.

Also, you can use classes like `lpds-border-radius-{size}-{position}` to apply border radius to one of the side:
- `top`
- `top-left`
- `top-right`
- `bottom`
- `bottom-left`
- `bottom-right`
- `left`
- `right`

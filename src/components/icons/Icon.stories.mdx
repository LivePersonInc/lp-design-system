# Icons

Each icon is a separate module.
It means that you have to import a specific module for the icon that you are going to use

For example, if you want to use `arrow-down` icon you have to import `icons/arrow-down.js` module to use it
```
// npm
import '@liveperson/design-system/icons/arrow-down';

<lp-arrow-down-icon />

// browser
<script src="icons/arrow-down.js"></script>

<lp-arrow-down-icon></lp-arrow-down-icon>
```

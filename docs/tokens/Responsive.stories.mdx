# Responsive

## How to use

### SCSS

```scss
@use '~@liveperson-design-system/tokens/scss/responsive';

.button {
  max-width: 100%;

  @media (min-width: responsive.$width-laptop) {
    max-width: 250px;
  }

  // or

  @include responsive.media(laptop) {
    max-width: 250px;
  }
}
```

### JS

```js
import styled from 'styled-components';

import { mixins } from '@liveperson-design-system/tokens/js/responsive';

const Button = styled.button`
  ${mixins.media('laptop')(`
    max-width: 250px;
  `)}
`;
```

### CSS

```html
<body class="lpds-responsive">
  <button>Button</button>
</body>

<style>
  @media (min-width: var(--lpds-breakpoints-laptop)) {
    button {
      max-width: 250px;
    }
  }
</style>
```

## Mixins

### SCSS

- `media($breakpoint, $up: true)` - to apply `@media` rule
  - `$breakpoint` - one of the available breakpoints or a value
  - `$up` - to switch between `@media (min-width: $breakpoint)` and `@media (max-width: $breakpoint)`
- `media-from-to($from, $to)` - to apply `@media` rule
  - `$from` - one of the available breakpoints or a value
  - `$to` - one of the available breakpoints or a value

### JS

- `media: (breakpoint: BreakpointArg, up: boolean = true) => (content: string): string`
- `mediaFromTo: (from: BreakpointArg, to: BreakpointArg) => (content: string): string`

## CSS helpers classes

### Classes

To hide elements simply use the `lpds-responsive-hidden` class
or one of the `lpds-responsive-hidden-{laptop|desktop|desktop-large}` classes for any responsive screen variation.

To show elements simply use the `lpds-responsive-visible` class
or one of the `lpds-responsive-visible-{laptop|desktop|desktop-large}` classes for any responsive screen variation.

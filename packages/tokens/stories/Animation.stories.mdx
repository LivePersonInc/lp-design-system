# Animation

These are a set of animation values.

## How to use

### SCSS

```scss
@use '~@liveperson-design-system/tokens/scss/animation';

@import '~@liveperson-design-system/tokens/scss/animation-keyframes';

.buttons {
  animation: animation.$animation-slide-up-fade-in;
}
```

### JS

```js
import { variables, keyframes } from '@liveperson-design-system/tokens/scss/animation';
```

### CSS

`animation.css` includes CSS variables, helpers classes and keyframes animations

```html
<link rel="stylesheet" href="https://unpkg.com/@liveperson-design-system/tokens/css/animation.css">
```

To use CSS variables just add `lpds-animation` class to the container

```html
<body class="lpds-animation">
  <button>Button</button>
</body>

<style>
  button {
    transition-duration: var(--lpds-animation-duration-fast);
  }
</style>
```

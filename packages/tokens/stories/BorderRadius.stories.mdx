# Border radius

Border-radius are used on most components that are used as containers.
Nested components should be smaller than the parent component.

## How to use

### SCSS

```scss
@use '~@liveperson-design-system/tokens/scss/border-radius';

.buttons {
  border-radius: border-radius.$border-radius-xxs;
}
```

### JS

```js
import { variables, functions, mixins } from '@liveperson-design-system/tokens/scss/border-radius';

const Button = styled.button`
  border-radius: ${variables.borderRadius.xxs};
`;

// or

const Button = styled.button`
  border-radius: ${functions.getBorderRadius('xxs')};
`;

// or

const Button = styled.button`
  ${mixins.borderRadius('xxs')}
`;
```

### CSS

`border-radius.css` includes CSS variables and helpers classes

```html
<link rel="stylesheet" href="https://unpkg.com/@liveperson-design-system/tokens/css/border-radius.css">
```

To use CSS variables just add `lpds-border-radius` class to the container

```html
<body class="lpds-border-radius">
  <button>Button</button>
</body>

<style>
  button {
    border-radius: var(--lpds-border-radius-xxs);
  }
</style>
```

## JS

### Functions

- `getBorderRadius` - to get border-radius by size (s, m, l, etc.) Example: `functions.getBorderRadius('xxs') => '4px'`

### Mixins

- `borderRadius` - returns border-radius rule by size (s, m, l, etc.). Example: `mixins.borderRadius('xxs') => 'border-radius: 4px;'`

## CSS helpers classes

### CSS variables

To use CSS variables like `--lpds-border-radius-xxs` you have to add `lpds-border-radiuss` class to the root tag.
In most cases this is `body` tag.

### Classes

To apply `border-radius` style just add `lpds-border-radius-{size}` class to the element.

Also, you can use classes like `lpds-border-radius-{size}-{position}` to apply border radius to one of the side:
- `top`
- `top-left`
- `top-right`
- `bottom`
- `bottom-left`
- `bottom-right`
- `left`
- `right`

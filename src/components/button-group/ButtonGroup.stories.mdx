# Button

<ArgsTable of={ButtonGroup} />

All buttons have to be provided as children elements

## Basic

<Canvas withSource={SourceState.OPEN}>
  <Story
    name="Basic"
    args={{
      theme: 'dark',
      size: 'default',
      buttons: ['Button1', 'Button2', 'Button3', 'Button4', 'Button5'],
    }}
  >
    {({ theme, size, buttons }) => (
      <lp-button-group
        key={buttons.toString()}
        theme={theme === 'dark' ? undefined : theme}
        size={size === 'default' ? undefined : size}
      >
        {buttons.map((button, index) => <button key={index}>{button}</button>)}
      </lp-button-group>
    )}
  </Story>
</Canvas>

## Default active

You can easily set the default active button by adding `active` class to the button item

<Canvas withSource={SourceState.OPEN}>
  <Story
    name="Default active"
    args={{
      theme: 'dark',
      size: 'default',
      active: 2,
      buttons: ['Button1', 'Button2', 'Button3', 'Button4', 'Button5'],
    }}
  >
    {({ theme, size, active, buttons }) => (
      <lp-button-group
        key={`${buttons.toString()}-${active}`}
        theme={theme === 'dark' ? undefined : theme}
        size={size === 'default' ? undefined : size}
      >
        {buttons.map((button, index) => (
          <button key={index} class={index === active - 1 ? 'active' : undefined}>{button}</button>
        ))}
      </lp-button-group>
    )}
  </Story>
</Canvas>

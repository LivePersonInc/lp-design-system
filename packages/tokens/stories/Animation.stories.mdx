# Animation

These are a set of animation values.

## How to use

### SCSS

```scss
@use '~@liveperson-design-system/tokens/scss/animation';

@import '~@liveperson-design-system/tokens/scss/animation-keyframes';

.button {
  animation: animation.$animation-slide-up-fade-in;
}
```

### JS

```js
import { variables, keyframes as LPDSLeyframes } from '@liveperson-design-system/tokens/scss/animation';

const slideUpFadeIn = keyframes`
  ${LPDSLeyframes.slideUpFadeIn}
`;

const Button = styled.button`
  animation: ${slideUpFadeIn} ${variables.animationSetting};
`;
```

### CSS

`animation.css` includes CSS variables, helpers classes and keyframes animations

```html
<link rel="stylesheet" href="https://unpkg.com/@liveperson-design-system/tokens/css/animation.css">
```

To use CSS variables just add `lpds-animation` class to the container

```html
<body class="lpds-animation">
  <button>Button</button>
</body>

<style>
  button {
    transition-duration: var(--lpds-animation-duration-fast);
  }
</style>
```

## CSS helpers classes

### CSS variables

To use CSS variables like `--lpds-animation-slide-up-fade-in` you have to add `lpds-animation` class to the root tag.
In most cases this is `body` tag.

### Duration

To apply `animation-direction` style just add `lpds-animation-duration-{duration}` class to the element.

Available durations:

- `fast`
- `normal`
- `slow`

### Timing function

To apply `animation-timing-function` style just add `lpds-animation-timing-function-{timing-function}` class to the element.

Available timing functions:

- `ease-in-quad`
- `ease-out-quad`
- `ease-in-out-quad`
- `ease-in-cubic`
- `ease-out-cubic`
- `ease-in-out-cubic`
- `ease-in-quart`
- `ease-out-quart`
- `ease-in-out-quart`

### Animation setting

To apply `animation` style with default value to use in all animations just add `lpds-animation-setting` class to the element.

### Animation

To apply `animation` style just add `lpds-animation-{animation-name}` class to the element.

Available animations:

- `slide-up-fade-in`
- `slide-up-fade-out`
- `slide-down-fade-in`
- `slide-down-fade-out`

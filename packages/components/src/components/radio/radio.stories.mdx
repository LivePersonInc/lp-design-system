`lp-radio` component always has to work with `lp-radio-group` component. For this, you don't need to import two modules.
`lp-radio-group` and `lp-radio` components place in one `radio` module

## Radio Group

export const LPRadioGroup = props => (
  <lp-radio-group {...props}>
    {Array(5).fill(null).map((element, index) => <lp-radio key={index} label={`Radio ${++index}`} value={index} />)}
  </lp-radio-group>
);

# InputHelper

<ArgsTable of={InputHelper} />

Helper text can be provided by `text` attribute or as a child

Example:
```html
<lp-input-helper text="Helper text" />
```
or
```html
<lp-input-helper>Helper text</lp-input-helper>
```

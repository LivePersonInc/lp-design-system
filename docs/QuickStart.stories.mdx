# Quick Start

## Installing

<TabsState>
  {packages.map(({ name, title }) => (
    <div key={name} id={name} title={title}>
      <Source language="bash" code={`yarn add ${name}`} />
    </div>
  ))}
</TabsState>

## How to use

### Setup

Before using `@liveperson-design-system/icons` or `@liveperson-design-system/components`,
you have to add the following script inside `head` tag

<TabsState>
  {packages.filter(({ key }) => (key === 'icons' || key === 'components')).map(({ key, name, title }) => (
    <div key={key} id={key} title={title}>
      <Source language="html" code={`<script src="https://unpkg.com/${name}/vendor.js"></script>`} />
    </div>
  ))}
</TabsState>

### Usage

You can use any LPDS component either in a browser or with a front-end framework

<TabsState>
  <div id="browser" title="Browser">
